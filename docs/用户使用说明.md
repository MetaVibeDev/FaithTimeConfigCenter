## 简介

FaithTime 配置中心是一个邀请码管理系统，用于管理和追踪平台的邀请码使用情况。通过本系统，您可以：

- 查看所有邀请码的状态和使用情况
- 将邀请码绑定到特定用户
- 追踪每个邀请码的推广效果
- 管理邀请码的启用/停用状态
- 查看用户详细信息

## 邀请码管理

登录成功后，您将进入邀请码管理主界面。

### 查看邀请码列表

主界面展示所有邀请码的详细信息：

| 列名         | 说明                                       |
| ------------ | ------------------------------------------ |
| **邀请码**   | 唯一的邀请码字符串                         |
| **状态**     | 显示邀请码是否已冻结（灰色标签表示已冻结） |
| **推广次数** | 该邀请码成功推广的用户数量                 |
| **绑定用户** | 邀请码绑定的用户邮箱或 ID                  |
| **绑定日期** | 邀请码绑定到用户的日期                     |
| **等级**     | 邀请码的推广等级                           |
| **操作**     | 可执行的操作按钮                           |

### 搜索邀请码

使用搜索功能快速定位特定邀请码：

1. 在页面顶部的搜索框中输入邀请码的全部或部分内容
2. 系统会实时过滤并显示匹配的结果
3. 清空搜索框可恢复显示所有邀请码

### 绑定用户

将邀请码绑定到特定用户：

1. 在邀请码列表中找到目标邀请码
2. 点击该行的"绑定用户"按钮
3. 在弹出的对话框中输入用户的邮箱地址
4. 点击"绑定"按钮完成操作

**绑定规则**：

- 每个邀请码只能绑定一个用户
- 绑定后会自动记录绑定日期
- 已绑定的邀请码在列表中会显示绑定的用户信息

### 查看推广用户

查看通过特定邀请码注册的所有用户：

1. 在邀请码列表中找到目标邀请码
2. 点击"推广次数"列中的数字（显示为蓝色可点击）
3. 系统会在右侧弹出侧边栏，展示该邀请码推广的所有用户列表

**推广用户列表包含**：

- 用户 ID
- 用户昵称
- 用户邮箱
- 注册日期
- 其他相关信息

> **提示**：点击列表中的用户可以查看该用户的详细信息。

### 冻结邀请码

停用某个邀请码，使其无法继续被使用：

1. 在邀请码列表中找到目标邀请码
2. 点击该行的"冻结"或"停用"按钮
3. 确认操作
4. 邀请码状态将变为"已冻结"，并在列表中显示灰色标签

**注意事项**：

- 冻结后的邀请码无法再被新用户使用
- 已冻结的邀请码仍可查看历史推广记录
- 冻结操作通常不可撤销，请谨慎操作

### 查看用户信息

查看用户的详细个人资料：

**方式一：从推广用户列表查看**

1. 打开邀请码的推广用户列表
2. 点击列表中任意用户
3. 系统会展开用户详细信息面板

**方式二：从绑定用户查看**

1. 将鼠标悬停在"绑定用户"列的用户 ID 上
2. 系统会显示用户信息预览卡片
3. 点击用户 ID 可查看完整的用户信息

**用户信息包含**：

- 基本信息（昵称、邮箱、ID）
- 账户状态
- 注册时间
- 会员等级
- 其他相关数据
